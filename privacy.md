# 개인정보처리방침 (Privacy Policy)

본 개인정보처리방침은 대성에너지(DAESUNG ENERGY)에서 제공하는 크롬 확장 프로그램 'DAESUNG Security'의 사용자 데이터 수집 및 처리 관행을 설명합니다.

## 1. 수집하는 개인정보의 항목

[크롬 확장프로그램 사용 시]
- 접속 IP, 브라우저 정보
- 방문 웹페이지 URL: 피싱·악성 사이트 여부 판정을 위해 접속하는 모든 웹페이지 주소가 실시간으로 전송·대조되며, 이 중 위험(블랙리스트)으로 판정된 URL과 출발 페이지(Referer) 주소만 서버에 보관됩니다.
- 입력 텍스트 모니터링: 개인정보(PII) 유출을 실시간 차단하기 위해 사용자가 입력하는 텍스트를 브라우저 내에서 모니터링합니다. 입력 원문은 서버로 전송되지 않으며, 탐지된 개인정보 유형·글자 수·이용 서비스 도메인 등 메타데이터만 서버로 전송됩니다.
- 첨부파일 콘텐츠: 개인정보 포함 여부 검증을 위해 첨부되는 문서·이미지 파일의 내용이 서버로 전송되어 실시간 분석됩니다.
- 서비스 이용 기록(접속 일시 등)

## 2. 개인정보 처리 목적
- 사내 보안 정책에 따른 외부 AI 서비스로의 기업 기밀 및 개인식별정보(PII) 유출 실시간 차단, 피싱 및 악성 URL 탐지 및 접근 격리

## 3. 개인정보의 보유 및 이용기간
- 수집된 로그: 6개월마다 자동 삭제

## 4. 개인정보의 안정성 확보 조치
- 통신 암호화: HTTPS 적용
- 접근 권한 관리: 관리자 계정(최소 권한 원칙 적용)
